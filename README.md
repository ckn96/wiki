# wiki of kaini

使用MDWiki创建的Wiki网页

## dev

1. install nodejs,use https://nodejs.org/en/
1. ```npm i```, install dependency
1. edit docs
1. run build js
1. run ```npm start```, start local dev server, use http://localhost:3000 access that

## build

```bash
npm run build
```

## deploy

with cloud foundry cli

```bash
cf push
```
